## საიტის შესაძლებლობები

- **მთავარი გვერდი**: მთავარ გვერდზე მომხმარებლებს აქვთ შესაძლებლობა, ღილაკზე კლიკით იხილონ ახალი ციტატა, დააკოპირონ იგი საკუთარ მოწყობილობაზე და გადააზიარონ საკუთარ ფეისბუქ ექაუნთზე სურვილისამებრ.
- **ფილტრი**: ფილტრში მომხმარებლებს აქვთ შესაძლებლობა, იხილონ ციტატელის ბაზაში არსებული ავტორების, თემატიკებისა და წყაროების სია, რომელზე დაკლიკებითაც, საიტი გადავა არჩეული ელემენტის კონკრეტულ ციტატებზე.
  ასევე, არის საძიებო ველი, რომელშიც შესაძლებელია სასურველი ავტორის, თემის ან წყაროს მოძიება.
- **ფილტრის რეზულტატები**: კონკრეტული ელემენტის არჩევის შემდეგ, ვებ–საიტი აჩვენებს არჩეულ ელემენტთან დაკავშირებულ ციტატებს.
- **ციტატების ბარათები**: თითოეულ ციტატას, ზედა მარჯვენა კუთხეში აქვს ღილაკი, რომელზე დაკლიკების შემდეგაც, იხსნება დამატებით ინფორმაციის პანელი. სადაც, მოცემულია არჩეულის გარდა დარჩენილი 2 ელემენტი, მაგალითად ავტორი => (წყარო, თემატიკა) და მომხმარებელს შეუძლია მოცემული ციტატის წყაროზე ან თემატიკაზე დაკავშირებული სხვა ციტატების ნახვაც.
- **ძიება**: ძიების კომპონენტი მომხმარებელს შესაძლებლობას აძლევს კონკრეტული ასოებით ან სიტყვებით მოიძიოს მისთვის სასურველი ციტატა.
  თუ საძიებო სისტემაში ჩაწერილი ასოები/სიტყვები ემთხვევა ციტატელის ბაზაში არსებულ ციტატებს, მომხმარებელი დაინახავს მას "Highlight" ამონათებული დიზაინით.
- **გენერატორი**: გენერატორის კომპონენტში მომხმარებელს შეუძლია აირჩიოს ავტორის, თემატიკისა და წყაროს ასარჩევი ველიდან მისთვის სასურველი ელემენტები. "გენერირება" ღილაკზე დაკლიკების შემდეგ, კი მომხმარებელი იხილავს მის არჩეულ ელემენტებთან დაკავშირებულ ციტატების დაჯგუფებულ ელემენტებს.
